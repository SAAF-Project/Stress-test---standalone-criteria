# Prompts

Domain-specific audit prompts and system instructions for AI audit agents.

## Subfolders

| Folder | Contents |
|---|---|
| [`system-instructions/`](system-instructions/) | CLAUDE.md / Copilot instructions / system prompts per AI tool |
| [`audit-use-cases/`](audit-use-cases/) | Per use-case prompt files (e.g. access-review, journal-entry testing) |
| [`conventions/`](conventions/) | Reusable prompt patterns and coding conventions |

## Contributing

- One file per use case or instruction set.
- Use Markdown for all prompt files.
- Name files in lowercase with hyphens (e.g. `access-review.md`).
- Include a brief header comment explaining the prompt's purpose and intended AI tool.
